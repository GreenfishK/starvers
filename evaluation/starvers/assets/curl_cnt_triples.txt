# Q1: Count triples per version
## GraphDB

curl -G "http://localhost:7600/repositories/ic_sr_ng_orkg" \
  --data-urlencode "query=SELECT ?g (COUNT(*) AS ?count) WHERE { GRAPH ?g {?s ?p ?o }} group by ?g order by ?g" \
  -H "Accept: application/sparql-results+json"
